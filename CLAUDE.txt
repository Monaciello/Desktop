# CLAUDE.txt - NixOS Config Quick Reference

## Structure
```
flake.nix
├── hosts/alice/             # SYSTEM: boot, services, hardware
│   └── secrets.yaml         # sops-nix encrypted (skarabox pattern)
├── home-manager/            # USER: apps, dotfiles
│   ├── modules/             # i3, kitty, neovim, xonsh
│   ├── packages/            # apps, cli, dev
│   └── programs/            # git, tmux
├── shells/                  # PROJECT: dev environments
├── docs/                    # keybindings, skarabox guides
└── .claude/                 # AI guardrails (AI.md, CLAUDE.md)
```

## Layer Rule
```
SYSTEM  (nixos-rebuild)  → hardware, services, boot
USER    (home-manager)   → apps, dotfiles, shell
PROJECT (nix develop)    → python, uv, dev tools
RUNTIME (.gitignored)    → .venv/, node_modules/
```

## Commands
```bash
sudo nixos-rebuild switch --flake ~/.config/nixos#alice
home-manager switch --flake ~/.config/nixos#sasha@alice
nix develop .#dev
nix flake check
```
